# Exchange Phishing Cleanup Script

This PowerShell script helps administrators quickly remove phishing emails from on-premises Microsoft Exchange environments (2010, 2016, 2019).

## Features

- Search and remove phishing emails
- Works with Exchange Management Shell
- Suitable for incident response scenarios

## Usage

1. Open Exchange Management Shell
2. Run the script:

```powershell
.\cleanup-phishing.ps1
```

- Follow the prompts
- Requirements
- On-premises Exchange (2010 / 2016 / 2019)
- Administrator privileges
